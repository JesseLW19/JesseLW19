Cheat Sheet
< > Surround each Attribute
!DOCTYPE HTML - declare at top of file
html - declare under DOCTYPE, and end of file after body
body - declare under html, and end of file before html
p /p - Start and end paragraph
h1 /h1 - Start and end header
br - break line
title /title - start and end page title
a href="link">Link name /a - Set link
