Cheat Sheet


Basic Structure

< > Surround each Attribute
!DOCTYPE HTML - declare at top of file
html - declare under DOCTYPE, and end of file after body
body - declare under html, and end of file before html
p /p - Start and end paragraph
h1 /h1 - Start and end header
br - break line
title /title - Start and end page title
a href="link">Link name /a - Set link

Image Attributes:

img src=link - set image
alt="alternative text"
width="num"
height="num"

Formatting

b - Bolded Text
i - Italicized Text
strong - Important Text
em - Emphasized Text
mark - Marked Text
small - Small Text
del - Deleted Text
ins - Inserted Text
sub - Subscript Text
sup - Superscript Text

Style Attribute

< tag > Tags are things like h1 or p
< tagname style="property:value;" >
< tag"background-color:color;" > Sets background color
< tag "color:color" > Sets text Color
< tag "font-family:family;" > Sets text font
< tag "font-size:300%;" > Sets text size
< tag "text-align:place (center left right);"> Aligns text
